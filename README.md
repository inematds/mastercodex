# Master Codex

> Curso completo de desenvolvimento agentic com **ChatGPT 5.5 Codex** — do zero ao SaaS publicado e vendável.

🌐 **Site:** [https://inematds.github.io/mastercodex/](https://inematds.github.io/mastercodex/)

---

## Sobre

Master Codex é um curso premium em formato HTML (texto + imagens), seguindo o padrão visual [INEMA.CLUB](https://inema.club).

**Promessa:** transformar você de "vibe coder" curioso em operador de uma fábrica de SaaS com agentes de IA — com seu primeiro cliente pagante ainda durante o curso.

**Projeto-âncora:** **InboxAI** — plataforma omnichannel de atendimento e vendas com agentes de IA, construída ao longo das Trilhas 3, 4 e 5.

---

- **IA:** OpenAI / Anthropic via Codex
- **Deploy:** Vercel + Convex Cloud
