## Como usar

### 1. Comece pelo `AGENTS.md`
Mesmo que você não vá usar o boilerplate InboxAI, leia o `AGENTS.md`. Os comentários explicam **por que** cada regra existe. Esse é o coração de trabalhar bem com agentes — escrever boas instruções.

### 2. Instale as 5 skills
Veja `skills/README.md`. Você copia a pasta de cada skill pra `~/.claude/skills/<nome>/` e elas ficam disponíveis em qualquer projeto.

### 3. Use o boilerplate InboxAI quando for vender pra cliente
Stack pronta. Documentação pronta. Você entrega em 2-3 dias o que demoraria 2-3 semanas do zero.

### 4. Verticalize
As pastas em `verticais/` te dão 4 nichos prontos pra prospectar. Pega a customização, ajusta um pouco e fecha contrato.

### 5. Renderize vídeos com Remotion
Quando precisar de um vídeo de lançamento, abra o pacote Remotion, troque os textos das composições e renderize. 30 minutos, vídeo profissional pronto.
